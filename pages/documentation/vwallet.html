Installation

Follow the steps below to install:
vwallet require PHP >= 8.0.:

  1. Clone repository from github: 
    git clone https://github.com/vantanto/vwallet.git
cd vwallet
  2. Install dependencies using composer 
    composer install
  3. Copy and Setup database in .env file 
    cp .env.example .env
  4. Generate key & Run migration, seeding & Start local developement 
    php artisan key:generate
php artisan migrate --seed
php artisan serve
  5. You can now access the server at http://localhost:8000
